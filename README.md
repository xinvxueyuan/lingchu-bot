# lingchu-nonebot-bot

## 项目简介

一个基于NoneBot2的QQ机器人项目

## 安装指南

1. 安装Python 3.13+
2. 安装OneBot实例
3. 安装nonebot2 cli
4. 创建nonebot2项目
5. 安装驱动器fastapi+~httpx+~websockets
6. 安装适配器nonebot2-adapter-onebot
7. 在插件目录下克隆本项目
8. 启动nonebot2项目
