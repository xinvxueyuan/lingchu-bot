# 5 分钟学 Markdown

## 标题

```
# H1 标题
## H2 标题
### H3 标题
#### H4 标题
##### H5 标题
###### H6 标题
```

## 文本格式

```
**加粗文本**
*斜体文本*
***加粗加斜体***
~~删除线~~
`行内代码`
```

## 链接与图片

```
[链接文本](https://example.com)
[带标题的链接](https://example.com "悬停标题")
![替代文本](image.jpg)
![带标题的图片](image.jpg "图片标题")
```

## 列表

```
无序：
- 项目 1
- 项目 2
  - 嵌套项

有序：
1. 第一项
2. 第二项
3. 第三项
```

## 引用

```
> 这是一个引用
> 多行内容
>> 嵌套引用
```

## 代码块

````
```javascript
function hello() {
  console.log("Hello, world!");
}
```
````

## 表格

```
| 表头 1 | 表头 2 | 表头 3 |
|--------|--------|--------|
| 行 1   | 数据   | 数据   |
| 行 2   | 数据   | 数据   |
```

## 分割线

```
---
或
***
或
___
```

## 任务清单

```
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个任务
```

## 转义字符

```
使用反斜杠进行转义： \* \_ \# \`
```

## 换行

```
在行末添加两个空格  
以创建换行。

或使用空行开始新段落。
```
