## 命令与群管理

当前可确认的用户侧能力集中在 Milky 群管理命令。命令由 `nonebot-plugin-alconna` 注册，默认启用命令起始符和命令分隔符。

## 成员禁言

主命令：

```text
禁言 @用户 [时长秒数] [原因]
```

默认值：

- 时长：`60` 秒。
- 原因：`违反群规「默认」`。

常见别名包括：

- `禁言用户`
- `禁言群成员`
- `禁言成员`
- `禁`
- `封禁`

## 成员解禁

主命令：

```text
解禁 @用户 [原因]
```

常见别名包括：

- `解禁用户`
- `解禁群成员`
- `解禁成员`
- `解封`
- `解除封禁`
- `解除禁言`

## 全体禁言

主命令：

```text
全员禁言
```

常见别名包括：

- `开启全体禁言`
- `全禁`
- `全禁言`
- `全体禁言`
- `开启全员禁言`
- `禁言群`

## 全体解禁

主命令：

```text
全体解禁
```

常见别名包括：

- `全员解禁`
- `关闭全体禁言`
- `解除全体禁言`
- `解禁全体`
- `解禁全员`
- `关闭全员禁言`
- `解禁群`

## 错误反馈

群管理命令会对 Milky 的网络异常和操作拒绝返回可读错误消息。未知异常仍会向外抛出，方便开发者发现未覆盖的问题。
