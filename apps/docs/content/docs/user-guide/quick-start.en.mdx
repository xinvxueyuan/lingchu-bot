## Quick Start

This page explains how to prepare Lingchu Bot locally. Before connecting to a real platform, prepare the account, network, and configuration required by NoneBot and the Milky adapter.

## Requirements

- Python 3.13
- uv
- git
- A runtime environment usable by the Milky adapter

## Get the code

```bash
git clone https://github.com/xinvxueyuan/lingchu-bot.git
cd lingchu-bot
```

## Install dependencies

```bash
uv sync --frozen
```

## Start

```bash
uv run python bot.py
```

`bot.py` initializes NoneBot, registers the Milky adapter, and loads the local plugin directory from `pyproject.toml`.

## Pre-run checklist

- Confirm that Python is version 3.13.
- Confirm that the Milky-side service and connection settings are ready.
- Confirm that Boolean values in NoneBot configuration use lowercase `true` / `false`, not `True` / `False`.
