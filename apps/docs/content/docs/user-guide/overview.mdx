## 用户指南概览

Lingchu Bot 当前更适合开发、测试和早期试用。文档会尽量描述仓库中已经能看到的能力，不承诺尚未验证的平台行为。

## 你可以期待什么

- 一个基于 NoneBot2 的机器人项目骨架。
- 一个默认注册 Milky 适配器的本地入口。
- 一组 Milky 群管理命令处理器。
- 面向后续功能扩展的配置、存储和数据库辅助模块。
- 基于 gettext 的运行时反馈翻译，当前提供简体中文和英文目录。

## 你需要准备什么

- Python 3.13。
- `uv`。
- 可连接的 Milky / NoneBot 运行环境。
- 对 NoneBot 配置文件和适配器连接方式的基本了解。

## 当前限制

- 项目仍处于 pre-alpha / development 阶段。
- `bot.py` 当前实际注册 Milky 适配器；其他适配器支持以插件元数据和后续实现为准。
- 命令覆盖范围仍较小，文档只列出当前仓库可确认的能力。
- 运行时 i18n 当前覆盖群管理反馈文本，不表示所有命令触发词都已本地化。
