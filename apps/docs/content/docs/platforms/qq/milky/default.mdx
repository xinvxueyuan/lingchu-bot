## Milky Default Implementation

The `default` package contains Milky handlers that rely only on the standard Milky API surface. These handlers run on any Milky implementation without implementation-specific extensions.

## Available commands

All commands listed in [QQ Commands](../commands) are available in the Milky default implementation, except:

- `发送群公告` / `send-announcement` — gated by implementation (see [LLBot](./llbot))
- `设置群头像` / `set-group-avatar` — not yet supported on Milky
- Remote management commands — OneBot V11 only

## Handler modules

| Module | Commands |
| --- | --- |
| `mute.py` | `禁言`, `解禁`, `全员禁言`, `全体解禁` |
| `member.py` | `踢出群成员`, `设置群名片`, `设置群头衔`, `设置群管理员`, `取消群管理员` |
| `block.py` | `拉黑`, `全局拉黑`, `删黑`, `全局删黑`, `清空黑名单`, `全局清空黑名单` |
| `profile.py` | `设置群名称` |
| `announcement.py` | `发送群公告` (LLBot only) |
| `lifecycle.py` | Bot lifecycle hooks (no user-facing command) |
| `menu.py` | Menu page handler |

## API differences from OneBot V11

| Operation | OneBot V11 | Milky |
| --- | --- | --- |
| Get member info | `bot.get_group_member_info()` returns `dict` | Returns `Member` model (use `.card`, `.nickname`) |
| Mute member | `bot.set_group_ban(group_id, user_id, duration)` | `bot.set_group_member_mute(group_id, user_id, duration)` |
| Mock pattern | `return_value={}` | `MagicMock(card="", nickname="")` |
