## Milky Protocol

Milky is a newer QQ adapter protocol. Lingchu Bot's Milky adapter lives in `handle/qq/adapters/milky/`. Set `LINGCHUAdapter = "~milky"` to switch from the default OneBot V11.

## Configuration

```dotenv
LINGCHUAdapter=~milky
```

## Implementation variants

| Implementation | Extension | Adds |
| --- | --- | --- |
| [default](./default) | — | Core group management |
| [LLBot](./llbot) | `llbot/` | Text-only group announcement |

<Callout type="warn" title="No remote management">

Remote management commands (`远程禁言`, `远程公告`, etc.) are OneBot V11 only. They are not available under Milky because the Milky protocol does not expose the same cross-group API surface.

</Callout>

## API notes

Milky APIs return pydantic `Model` objects, not `dict`. Access fields with attribute access, not `.get()`:

```python
member = await bot.get_group_member_info(group_id=gid, user_id=uid)
card = member.card  # not member.get("card")
```

Some API names differ from OneBot V11:

| Capability | OneBot V11 | Milky |
| --- | --- | --- |
| Mute member | `set_group_ban(group_id, user_id, duration)` | `set_group_member_mute(group_id, user_id, duration)` |
